# 🔫 todotodo

## Description
칭찬 리액션 체크리스트

🔗 Service : 
``
https://todotodo-one.vercel.app/
``  

## ✅ Prerequisites
- Node.js 20.x
- Yarn

## Tech Stack
- React
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase
- Yarn

## 📦 Installation

```bash
$ yarn install
```

## 🏃 Running the app
```bash
$ yarn
$ yarn dev

$ yarn lint
$ yarn lint --fix
```
